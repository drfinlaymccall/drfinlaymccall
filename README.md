# Hi, I'm Finlay

I build tools that help humans and AI agents learn together. My background is in education — 20+ years of teaching, training, and curriculum design — and my PhD explored how we can apply those principles to machine education: structured approaches to improving what autonomous agents can do.

Right now I'm working on **Machine Education**, a framework for conducting educational needs analysis on AI agents. It treats agent capability development the way a good curriculum designer treats learner development — with structured assessment, evidence-based objectives, and iterative refinement.

## What I'm interested in

- Human-AI collaboration as a design discipline, not just a prompting technique
- Curriculum frameworks applied to agent capability development
- The gap between what agents *can* do and what they *reliably* do — and how structured learning closes it
